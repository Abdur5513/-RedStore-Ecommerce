# RedStore - eCommerce Website

A modern and fully responsive eCommerce website built 
with pure HTML, CSS and JavaScript. No frameworks, 
no libraries — just clean code.

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla)

## ✨ Features
- Fully responsive design (mobile, tablet, desktop)
- Product listing pages
- Shopping cart UI
- Hero banner section
- Clean and modern UI
- Fast loading — no frameworks used
